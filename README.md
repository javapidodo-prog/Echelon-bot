# Echelon 🎯

**Echelon** — это высокопроизводительный Telegram-бот и агрегатор объявлений, предназначенный для мониторинга выгодных предложений на Avito. Система анализирует рынок, автоматически рассчитывает медианную цену для заданных товаров и мгновенно уведомляет пользователей о появлении позиций со значительной скидкой.

## 🚀 Ключевые возможности

- **Anti-detect Scraping**: Обход анти-бот систем Avito с использованием `Playwright` (скрытие автоматизации, поведенческий скролл, сохранение сессий).
- **Динамическая аналитика**: Автоматический расчет медианной цены по рынку на основе исторических данных. Выгода считается не от "желаемой цены", а от реальной картины рынка.
- **Гибкая настройка**: Отслеживание по ключевым и стоп-словам, ограничение по цене, фильтрация мусора и запчастей.
- **Многопользовательская архитектура**: Разделение на Free и Premium уровни (разные интервалы опроса и лимиты на отслеживаемые модели).
- **Admin Dashboard**: Встроенная легкая веб-админка для мониторинга статистики, точности срабатываний и управления конфигурацией.

## 🛠 Технологический стек

- **Язык**: Python 3.12+
- **Telegram API**: `aiogram 3.x`
- **Scraping**: `Playwright` (async API)
- **База данных**: SQLite + `aiosqlite` (асинхронное взаимодействие)
- **Конфигурация**: `pydantic-settings`
- **Архитектура**: TaskGroup для управления фоновыми задачами, Graceful Shutdown.

## ⚙️ Установка и запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python 3.12 или новее. Клонируйте репозиторий и создайте виртуальное окружение:

```bash
git clone https://github.com/yourusername/echelon.git
cd echelon
python -m venv .venv
source .venv/bin/activate  # Для Windows: .venv\Scripts\activate
```

### 2. Установка зависимостей
Установите пакетный менеджер `uv` (если еще не установлен), после чего скачайте зависимости и бинарники браузера для Playwright:

```bash
# Установка зависимостей через uv (использует pyproject.toml)
uv sync

# Установка браузера для парсинга
uv run playwright install chromium
```

### 3. Настройка конфигурации
Скопируйте файл примера конфигурации и заполните его:

```bash
cp .env.example .env
```
В файле `.env` **обязательно** укажите:
- `BOT_TOKEN`: токен вашего бота (получить у [@BotFather](https://t.me/BotFather)).
- `ADMIN_USER_ID`: ваш Telegram ID для доступа к админ-командам и веб-дашборду.

Запустить бота и панель администратора можно вручную:

```bash
uv run bot.py
# Отдельный терминал
uv run admin_dashboard.py
```

Веб-панель по умолчанию доступна на `http://127.0.0.1:8080`.

## 📂 Структура проекта

```text
Echelon/
├── bot.py                # Точка входа, запуск бота и фоновых задач
├── scraper.py            # Логика парсинга (Playwright + Pydantic)
├── tasks.py              # Фоновые процессы (Scraping, Уведомления, Очистка)
├── database.py           # Асинхронное взаимодействие с SQLite
├── config.py             # Настройки проекта (Pydantic Settings)
├── models_config.py      # Пресеты отслеживаемых моделей и стоп-слова
├── handlers/             # Обработчики команд и кнопок Telegram
├── utils/                # Вспомогательные функции, клавиатуры, middlewares
├── templates/            # HTML шаблоны для Admin Dashboard
└── docs/                 # Документация (инструкции по деплою на VPS)
```

## 📝 Админские команды

Доступны только пользователю с ID, указанным в `ADMIN_USER_ID`:
- `/validation` — статистика работы алгоритма (уведомления, точность).
- `/models_list` — список всех сконфигурированных моделей.
- `/set_median "Название" 10000` — ручная установка медианной цены.
- `/recalculate_medians` — принудительный пересчет медиан на основе БД.
- `/export_validation` — экспорт лога работы алгоритма в CSV.
- `/reports` — просмотр жалоб пользователей на ошибки алгоритма.
